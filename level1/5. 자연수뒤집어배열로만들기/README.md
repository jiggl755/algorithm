1.이해
====
자연수 N을 받은뒤 거꾸로 뒤집어 배열에 넣어서 리턴한다.
12를받은면 [2,1]
20005를받으면 [5,0,0,0,2]를 리턴

2.계획
===
n을 하나하나쪼개서 배열에들어가게한다음에 reverse 하면되는거아닌가?

3.실행
====
test('12', () => {
        expect(solution(12)).toBe([2,1]);
    });
test('2005', () => {
        expect(solution(2005)).toBe([5,0,0,2]);
    });
