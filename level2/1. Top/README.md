1.이해
====
탑의 꼭대기에 신호를 송수신하는 장치가있는 탑들이있다.
발사한 신호는 신호를 보낸탑보다 높은탑에서만 수신
한번수신된 신호는 다른탑으로 송신x
그래서 각각탑의 높이를 가지고있는 height 변수를 매개변수로주면
누구의 신호를받았는지 저장하느 배열을 뱉어내자
