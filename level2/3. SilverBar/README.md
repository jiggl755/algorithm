===

)를 만나면 하나씩 더해주고 스택에서는 뺀다

즉 arr에서 하나씩빼서 스택에서 넣다가, 다음글자가 ()면 지금스택의 length만큼 res에 더하고
)면 +1을하고 스택에서 하나 빼고
하는식으로한다
